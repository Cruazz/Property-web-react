# Property-web-react: Cruaz Hotel & Spa Frontend

A luxury-oriented, high-fidelity hotel and resort website template. Built with modern web standards and designed to deliver an unmatched first impression.

## 🚀 Technologies

- **React 19** with **TypeScript**
- **Vite** as build tool
- **Tailwind CSS v4** for flexible premium styling
- **Lucide React** for modern UI icons

## 🏛 Architecture

```
src/
├── components/   # UI elements, common layout wrappers, carousels
├── data/         # Mock data for Locations, Articles, Team, Projects
├── pages/        # All 7 core views
└── types/        # TypeScript interfaces and definitions
```

## 🛠 Setup & Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run local dev server:
   ```bash
   npm run dev
   ```

3. Build production bundle:
   ```bash
   npm run build
   ```
